Minecraft Chest Carrying Mod

Shift + right click to hold the chest.
Shift + right-click again to set up the chest.

Shift＋右クリックでチェストを持つ。
持った状態で再度Shift＋右クリックで設置できる。

But now, for some reason, Shift+right click only responds to some blocks (I have confirmed that it responds to chests, furnace, and jukeboxes).
ただ、現在はなぜかShift＋右クリックをしても一部のブロックにしか反応しない（チェスト、かまど、ジュークボックスには反応することを確認済み）
